Diese WebApp wurde für Mobil Geräte entwickelt und kann nicht von einem Laptop oder Desktop verwendet werden. Anwendung:
1.) Legen Sie das Gerät auf eine ebene Oberfläche.
2.) Stellen Sie die Lautstärke des Gerätes auf 100%.
3.) Wählen Sie über "Durchsuchen"/"Datei auswählen"/o.Ä. ein Lied aus.
4.) Das Lied beginnt automatisch direkt zu spielen. Ist das Display auf dem Tisch pausiert es. Liegt das Handy auf dem Rücken, spielt das Liede weiter.
5.) Drehen sie das Handy im Uhrzeigersinn um die Lautstärke zu erhöhen und gegen den Uhrzeigersinn um die Lautstärke zu reduzieren.
